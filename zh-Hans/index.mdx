title: "介绍"
description: "Hermai 是一个由社区驱动、专为 AI agent 打造的 schema registry。"
---

Hermai 是一个 schema registry,由开发者贡献网站 API schema,AI agent 则负责使用。将 agent 指向 registry 中的任何域名,即可获得完整的可调用端点列表 — URL、HTTP method、headers、参数 — 一切准备就绪可直接执行。无需浏览器、无需解析 HTML、不浪费 token。

可以把它想象成**网站 API 版的 HuggingFace**。

## 工作方式

<CardGroup cols={2}>
  <Card title="用 CLI 贡献" icon="terminal">
    使用发现工具组(`hermai detect`、`probe`、`extract`、`intercept`)记录网站端点,然后执行 `hermai registry push` 将其新增到目录。
  </Card>
  <Card title="从 registry 查询" icon="magnifying-glass">
    调用 `GET /v1/catalog/{domain}` 获取该站点所有已知端点。平台提供社区贡献的 schema — 即时、结构化、零发现成本。
  </Card>
</CardGroup>

<Tip>
  **在 AI agent 里使用?** 通过 `npx skills add hermai-ai/hermai-skills` 安装 [Hermai skills](https://github.com/hermai-ai/hermai-skills) — Claude Code、Codex、Cursor 以及其他 agent 就会知道如何调用 registry 并代替你贡献 schema。
</Tip>

## 开源

[hermai-cli](https://github.com/hermai-ai/hermai-cli) 以 AGPL-3.0 开源。你可以在本地运行完整的发现引擎,无需注册账号。平台 registry 是构建在之上的社区层 — 浏览、搜索并使用其他人贡献的 schema。
