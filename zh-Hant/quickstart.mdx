## 1. 註冊

到 [hermai.ai/register](https://hermai.ai/register) 建立免費帳號。不需要信用卡 — registry 完全免費使用。

## 2. 建立 API key

進入 **Dashboard > API Keys** 並建立新的 key。複製它 — 在需要驗證的請求中會用到。

```
hm_sk_abc123...
```

匿名請求也可以用(每小時 5 次),但 API key 提供每小時 50 次、每天 500 次的額度。

## 3. 查詢 catalog

查詢某個網域的所有已知端點:

```bash
curl -H "Authorization: Bearer hm_sk_abc123..." \
     -H "X-Hermai-Intent: looking up product endpoints on allbirds.com to build a price comparison agent" \
     "https://api.hermai.ai/v1/catalog/allbirds.com"
```

回應包含社群發現的每一個端點與 action:

```json
{
  "domain": "allbirds.com",
  "endpoints": [
    {
      "name": "shopify_products",
      "method": "GET",
      "url": "https://www.allbirds.com/products.json",
      "description": "Shopify store product catalog"
    },
    {
      "name": "add_to_cart",
      "method": "POST",
      "url": "https://www.allbirds.com/cart/add.js",
      "headers": { "Content-Type": "application/json" },
      "kind": "api_call",
      "params": [
        { "name": "id", "source": "body", "required": true },
        { "name": "quantity", "source": "body", "required": true }
      ]
    }
  ]
}
```

## 4. 呼叫端點 — 兩種方式

### 方式 A:自己呼叫網站(免費)

用 catalog 回應裡的 URL、method、headers 自行呼叫網站:

```bash
curl -X POST "https://www.allbirds.com/cart/add.js" \
     -H "Content-Type: application/json" \
     -d '{"id": 41397031600208, "quantity": 1}'
```

不用瀏覽器、不用 proxy、Hermai 也不在請求路徑上。免費。

### 方式 B:讓 Hermai 代為呼叫(每次成功 1 credit)

當你不想自己寫 HTTP / proxy / cookie / 簽章程式碼,或網站有 anti-bot 保護(x.com、booking.com、instagram.com 等)無法吃純呼叫時,改走 [cloud fetch](/zh-Hant/concepts/cloud-fetch)。

```bash
# 用 URL — 適合本來就吃純 HTTPS 的公開 API
curl -X POST https://api.hermai.ai/v1/fetch \
     -H "Authorization: Bearer hm_sk_..." \
     -H "Content-Type: application/json" \
     -d '{"url": "https://api.github.com/users/torvalds"}'

# 用名稱 — anti-bot 網站必須用這條
curl -X POST https://fetch.hermai.ai/v1/fetch \
     -H "Authorization: Bearer hm_sk_..." \
     -H "Content-Type: application/json" \
     -d '{
       "site": "x.com",
       "endpoint": "user_profile",
       "params": {"variables": "{\"screen_name\":\"jack\"}", "features": "{…}"}
     }'
```

兩種形態都回乾淨投射過的 JSON。完整參考見 [Cloud fetch](/zh-Hant/concepts/cloud-fetch)。

## 5. 瀏覽 registry

在 [hermai.ai/schemas](https://hermai.ai/schemas) 瀏覽,或透過 API:

```bash
# 列出所有 schema
curl "https://api.hermai.ai/v1/schemas"

# 依分類篩選
curl "https://api.hermai.ai/v1/schemas?category=commerce"

# 僅顯示已認證的 schema
curl "https://api.hermai.ai/v1/schemas?verified=true"

# 瀏覽 intent 分類樹
curl "https://api.hermai.ai/v1/categories"
```

## 6. 貢獻 schema

發現缺少的網站?用 CLI 記錄其端點並推送 schema:

```bash
# 安裝 CLI
go install github.com/hermai-ai/hermai-cli/cmd/hermai@latest

# 分類網站並找出公開路徑
hermai detect https://example.com
hermai wellknown example.com

# 從詳細頁面擷取內嵌資料
hermai probe --body https://example.com/products/123 | hermai extract

# 捕獲動態頁面的 XHR 流量(搜尋、購物車等)
hermai intercept https://example.com/search?q=test

# 撰寫 schema.json(參見 /zh-Hant/concepts/schemas),然後推送
hermai registry push schema.json
```

<Tip>
  **使用 Claude Code、Codex 或 Cursor?** 執行 `npx skills add hermai-ai/hermai-skills`,你的 agent 就會代替你完成探索與推送。詳情參見 [hermai-skills repo](https://github.com/hermai-ai/hermai-skills)。
</Tip>

每貢獻一個網站可獲得 **+50 credits**。Credits 可用在 [cloud-fetch](/zh-Hant/concepts/cloud-fetch) 呼叫上 — 每次貢獻都是為日後對 registry 的雲端請求充值。
